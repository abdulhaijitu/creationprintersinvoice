

## পরিকল্পনা: মোবাইলে `/` পেইজে টাইলস মেনু ডিফল্ট ভিউ

### বর্তমান অবস্থা
- `/` রাউটে `Dashboard` কম্পোনেন্ট রেন্ডার হয় (AppLayout এর ভেতর)
- `MobileSidebarTiles` একটি ফুল-স্ক্রিন ওভারলে যা শুধুমাত্র হ্যামবার্গার মেনু (≡) ক্লিক করলে দেখায়
- ইউজার চাচ্ছেন: মোবাইলে `/` ওপেন করলেই সরাসরি টাইলস গ্রিড দেখাবে, কোনো মেনু বাটন চাপার দরকার নেই

### পরিবর্তন

**ফাইল: `src/pages/Dashboard.tsx`**

ড্যাশবোর্ড কম্পোনেন্টের শুরুতে মোবাইল চেক যোগ:
- `useIsMobile()` হুক ব্যবহার করে মোবাইল ডিটেক্ট
- মোবাইলে: ড্যাশবোর্ড কন্টেন্টের বদলে `MobileSidebarTiles`-এর মতো একটি টাইলস গ্রিড রেন্ডার করবে (ইমেইজের মতো দেখতে)
- ডেস্কটপে: বর্তমান ড্যাশবোর্ড অপরিবর্তিত

**ফাইল: `src/components/layout/AppLayout.tsx`**

- মোবাইলে হোম পেইজে (`/`) থাকলে: টপবার এবং বটম নেভ হাইড করা হবে না, তবে ড্যাশবোর্ড পেইজ নিজেই টাইলস দেখাবে
- ব্যাক বাটন লজিক আগের মতোই কাজ করবে: সাব-পেইজে ← ক্লিক → `/` তে ফিরে আসবে → টাইলস দেখাবে

### মোবাইল ফ্লো

```text
স্টেপ ১: / পেইজ ওপেন (মোবাইল)
┌──────────────────────────────┐
│ [≡]                    [...] │  ← টপবার
├──────────────────────────────┤
│ MAIN                         │
│ [Dashboard] [Invoices] [Pay] │
│ [Quotes]  [PriceCalc] [Chal] │
│                              │
│ BUSINESS                     │
│ [Customers] [Vendors] [Exp]  │
│                              │
│ HR & OPS                     │
│ [Employees] [Attend] [Salary]│
└──────────────────────────────┘

স্টেপ ২: "Invoices" টাইলে ক্লিক → /invoices পেইজ
┌──────────────────────────────┐
│ [←]                    [...] │
├──────────────────────────────┤
│  (Invoices কন্টেন্ট)         │
└──────────────────────────────┘

স্টেপ ৩: ← ক্লিক → / পেইজে ফিরবে → আবার টাইলস
```

### প্রযুক্তিগত বিবরণ

**`src/pages/Dashboard.tsx`** এ পরিবর্তন:
- ফাইলের শুরুতে `useIsMobile()` চেক
- মোবাইলে early return করবে একটি টাইলস গ্রিড কম্পোনেন্ট দিয়ে
- টাইলস গ্রিড `sidebarNavGroups` থেকে আইটেম নেবে (যাতে পারমিশন ফিল্টারিং কাজ করে)
- প্রতিটি টাইলে ক্লিক করলে `navigate(item.url)` কল হবে
- ডেস্কটপে পুরো ড্যাশবোর্ড আগের মতোই থাকবে

**`src/components/layout/AppLayout.tsx`** এ পরিবর্তন:
- মোবাইলে হোম পেইজে হেডার থাকবে কিন্তু `SidebarTrigger` (≡) দিয়ে ফুল-স্ক্রিন ওভারলে টাইলস ওপেন করার অপশনও থাকবে
- বটম নেভ মোবাইল হোম পেইজে হাইড করা যেতে পারে (যেহেতু টাইলসই নেভিগেশন)

এই অ্যাপ্রোচে কোনো নতুন রাউট বা লেআউট পরিবর্তন লাগবে না — শুধু Dashboard কম্পোনেন্ট মোবাইলে ভিন্ন UI রেন্ডার করবে।
